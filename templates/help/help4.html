HOW TO RESET PASSWORD

While logging into the sytem,you might have forgotten your password. The system allows you to reset the password. By clicking on the Reset password below the 'password' entry, you are requested to provide an email that can be used to send the one time new password. After providing the email, a one time password is sent to the email provided. You can use the one time password to log into tthe system then use the 'change password' button to change the password to a more memorable password.
